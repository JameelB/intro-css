CSS Selectors

Selectors are patterns used to select the element you want to style. There are multiple selectors that we can use to target a specific element(s).

Using the Element Tag as a Selector

Element tags can be used as a selector.
When using an element tag, all of the elements with that specific tag will have the style rule applied to them.
This is specified in css as [tagName] { ...[style rules]... }
For Example:

p { color: blue; }

Using Classes as a Selector

HTML elements can be assigned a class attribute. Classes can be used to identify more than one element.
When a style rule is applied to a class, it will apply that rule to all HTML elements that has that specific class value.
In CSS, class selectors are specified by .[className]
For Example:

.sampleClass { color: black; }

Using IDs as a Selector

HTML elements can be assigned an id attribute. IDs should be unique to each element and can only be used to identify one element.
When a style rule is applied to an id, it will that rule to that specific HTML element that has that specific id value.
In CSS, id selectors are specified by #[idName]
For Example:

#sampleId { color: black; }
